Item API¶

An Item is a physical object in the world. Every item contains a SubItem structure for visual representation. It adds physics, positions, and physical statuses.

Properties¶

Property Type Description
position_x int X coordinate in the world (multiplied by 1000).
position_y int Y coordinate in the world (multiplied by 1000).
velocity_x int Current horizontal velocity.
velocity_y int Current vertical velocity.
impulse_x int Applied horizontal impulse.
impulse_y int Applied vertical impulse.
is_in_the_air bool True if the item is not touching the ground.
is_active bool True if the item is currently active in the game.
is_obstacle bool True if the item acts as a physical barrier.
has_weapon bool True if the character is currently holding a weapon.
weapon Item Returns the Item object for the weapon currently held.
is_vulnerable bool True if the item can be hit.
is_knocked_down bool True if the item is in a knocked-down state.
was_hit_from_right bool True if the last hit came from the right side.

Methods¶

get_nearest_opponent(in_same_line, forward_check, backward_check)¶

Finds the closest enemy relative to this item.

try_spawn_item(pack, name, anim, mode, off_x, off_y, vel_x, vel_y)¶

Spawns a new item into the world. Useful for projectiles or dropped items.

update_position_and_check_world_collisions()¶

Updates the item's position based on velocity and handles wall/floor collisions.

copy_position_to(other_item_id, offset_x, offset_y)¶

Synchronizes another item's position with this one.
