Scripting Examples¶

Here are some real-world examples taken from the Test chronicle pack to help you understand how to use the API.

Custom Blocking Logic¶

This script implements a manual block that reduces damage and handles knockback.

-- Block.lua
character = game.this_character

if (character.check_being_grabbed()) then
    character.is_blocking = false
    return
end

character.is_blocking = true

-- Reduce damage if hit while blocking
if (character.is_being_hit) then
    character.damage_taken = character.damage_taken / 6
    if (character.damage_taken < 1) then
        character.damage_taken = 1
    end

    -- Manual knockback
    character.impulse_x = 196
    character.is_facing_right = character.was_hit_from_right
    character.health = character.health - character.damage_taken

    -- Clear hit status so we don't fall down
    character.is_being_hit = false
    character.is_knocked_down = false
end

Spawning Projectiles¶

Example of Ryu spawning a Hadoken fireball at a specific animation frame.

-- Ryu Hadoken.lua
local character = game.this_character

if (character.current_animation_frame == 12) then
    -- packName, entityName, animationId, mode, offX, offY, velX, velY
    character.try_spawn_item("Test chronicle", "Ryu", 47, -1, -2750, -2750, -250, 0)
end

return true

AI & Targeting¶

Joe Musashi checking for the nearest opponent to decide between a melee attack or a shuriken.

-- Joe Shuriken.lua
local character = game.this_character

if (character.current_animation_id == 29 and character.current_animation_frame == 0) then
    -- Check for someone nearby in the same line
    local nearest_opponent = character.get_nearest_opponent(true, true, false)

    if (nearest_opponent ~= nil) then
        local distance = math.abs(character.position_x - nearest_opponent.position_x)

        -- If too close, switch to a different animation (melee)
        if (distance < 3000) then
            character.set_animation(30)
            character.play_sound("Test chronicle", "kunai_hit", 2)
            return true
        end
    end

    -- Otherwise, throw shuriken
    character.try_spawn_item("Test chronicle", "Shuriken", 0, -1, -750, 3000, 524, 0)
end
