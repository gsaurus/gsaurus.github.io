Enemy API¶

An Enemy is a specialized Character that is controlled by AI logic. It contains and extends all properties and methods from the Character API.

Properties¶

Property Type Description
is_alive bool True if the enemy is active and has health remaining.
mode int The current AI state.
mode_timer int Time remaining in the current AI state.
moving_direction int Direction the enemy intends to move.
distance int Target distance from the player to maintain.
movement_speed int Base movement speed of the enemy.
enemy_status EnemyStatus Enum representing the overall status of the AI.
enemy_agressiveness int How aggressively the enemy attacks (higher = more frequent).
homing_target_id int ID of the specific player this enemy is targeting.
homing_timer int Time remaining before the enemy switches targets or retargets.

Character-Specific AI Values¶

Some enemies have specific logic attached to these variables. The values can be overridden to change behavior patterns.

Property Type Description
enemy_specific_c0 int AI logic parameter 1.
abadede_specific_c1 int Logic parameter used specifically by Abadede.
abadede_specific_c2 int Logic parameter used specifically by Abadede.
enemy_specific_c3 int AI logic parameter 2.
enemy_specific_c4 int AI logic parameter 3.
enemy_specific_counter_c5 int Counter used in specific enemy modes.
shiva_specific_b0 int Logic parameter used by Shiva.
biker_special_counter_cb int Counter used by bikers.
is_biker_riding_bike_ca bool True if the biker is currently on a motorcycle.
use_biker_pipe bool True if the biker intends to swing a pipe.

Probability & AI Overrides¶

These properties are used by the AI engine to decide what to do next based on the enemy data.

Property Type Description
override_ai_data_e0 int Manual override parameter 1.
override_ai_data_e1 int Manual override parameter 2.
override_ai_data_e2 int Manual override parameter 3.
override_ai_data_e3 int Manual override parameter 4.
override_ai_data_e4 int Manual override parameter 5.
pack_name string The name of the content pack this enemy belongs to.

Read-Only AI Mode Configuration¶

These properties represent the raw AI data from the game's configuration files (often extracted directly from ROM mapping).

Property Type Description
probability_mode_e0 int Chance to switch to AI Mode 0.
min_timer_mode_0_e6 int Minimum time spent in AI Mode 0.
random_timer_mode_0_e7 int Random variance for AI Mode 0 timer.
min_speed_mode_1_ea int Minimum speed while in AI Mode 1.
random_speed_mode_1_eb int Random variance for AI Mode 1 speed.

Methods¶

return_to_standing()¶

Resets the enemy to its default standing state.

Enemies also rely on the inherited methods from Character and Item for actions like checking hits, setting animations, and applying throw damage.
