SubItem API¶

SubItems are the base components of all game objects, handling animations and basic properties.

Properties¶

Property Type Description
name string The internal name of the entity.
pack_name string The name of the content pack this entity belongs to.
is_enabled bool True if the object exists and is enabled.
is_facing_right bool The horizontal orientation of the sprite.
direction_signal int Returns 1 if facing right, -1 if facing left.
palette_pack int The numerical hash of the content pack for this palette.
palette_line int The color palette index currently in use.
next_animation_id int ID of the next animation scheduled to be played.
next_animation_frame int The frame index to start the next animation from.
current_animation_id int ID of the currently playing animation.
current_animation_frame int The current frame index within the animation.
is_animation_over bool True if the current animation has finished playing.
object_mode int The internal logic mode of the object.

Methods¶

set_animation(id, start_frame)¶

Sets the current animation and optionally the starting frame.

play_sound(pack, name, channel, pitch)¶

Plays a sound effect attached to this object.

clear_referenced_items()¶

Clears any items currently linked or held by this object.
