Streets of Rage New Era - Lua Scripting API¶

Welcome to the Lua Scripting API documentation for Streets of Rage New Era!

This documentation provides information on how to create and modify game logic using Lua scripts. Whether you want to add new moves, create custom enemy AI, or manipulate game objects, you can do it with Lua.

🛠️ Key Features¶

  • Full Control: Manipulate players, enemies, and items directly.
  • Easy Integration: Seamlessly call game functions and access engine properties.
  • Rapid Iteration: Modify scripts without recompiling the game.
  • Friendly Syntax: Uses a simple, snake_case mapping from the C# engine.

📂 Where to find scripts?¶

Scripts are typically located in the Scripts folder of your content pack. For example: Editor/Your Pack Name/Scripts/MyAwesomeScript.lua
