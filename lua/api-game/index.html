Game API (game)¶

The game object is the global entry point for all game logic scripts.

Properties¶

Property Type Description
current_frame int The current frame of the game engine.
global_timer int The total number of elapsed frames since the start.
level_id int The numerical ID of the current level.
scene_id int The numerical ID of the current scene/wave.
is_in_slow_motion bool True if the game is currently in slow motion.
slow_motion_frames_left int Remaining frames for the slow motion effect.
is_current_player bool True if the current script context is a player.
is_current_enemy bool True if the current script context is an enemy.
number_of_active_enemies int Count of enemies currently active in the scene.
number_of_alive_enemies int Count of enemies that are still alive.
number_of_free_sub_items int Number of available sub-item slots.
this_character Character The character (Player or Enemy) currently running this script.
this_player Player The player currently running this script (null if not a player).
this_enemy Enemy The enemy currently running this script (null if not an enemy).
this_item Item The item currently running this script.

Methods¶

get_player(id)¶

Returns a Player object by its entity ID.

get_enemy(id)¶

Returns an Enemy object by its entity ID.

get_character(id)¶

Returns a Character object (Player or Enemy) by its entity ID.

get_item(id)¶

Returns an Item object by its entity ID.

get_sub_item(id)¶

Returns a SubItem object by its entity ID.

get_free_enemy(consider_players_count)¶

Returns the first available enemy slot.

get_free_item()¶

Returns the first available item slot.

get_free_sub_item()¶

Returns the first available sub-item slot.

get_pack_hash(pack_name)¶

Returns the numerical hash of a content pack name.

set_clock(seconds)¶

Sets the level timer to the specified value in seconds.

set_slow_motion(frames)¶

Activates a slow-motion effect for a specified duration in frames.

reset_clock()¶

Resets the level timer to its initial value.

get_potential_score()¶

Calculates the current total score, including any end-of-level bonuses.

update_alternative_palette()¶

Updates the alternative palette for specific objects based on game rules.

schedule_go_forward_effect()¶

Triggers the "GO!" indicator on screen.

schedule_music(music_name, pack_name)¶

Schedules a background music track to be played. If pack_name is omitted, the current game pack is used.

is_next_wave_non_scrolling()¶

True if the next wave of enemies won't require screen scrolling.

get_number_of_active_players()¶

Returns the number of players currently active in the game.

kill_all_enemies(exception_id)¶

Immediately defeats all active enemies, optionally excluding one by its ID.

advance_level()¶

Immediately advances the game to the next level.

advance_wave()¶

Advances the game to the next wave of enemies.
