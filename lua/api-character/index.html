Character API¶

Every Character is built on top of an Item. It adds concepts of life, damage, and combat relationships.

Characters include both Players and Enemies. They inherit all properties and methods from the Item API.

Properties¶

Property Type Description
is_player bool True if this character is a player.
is_enemy bool True if this character is an enemy.
display_name string The name shown in the UI.
health int Current health points.
maximum_health int Maximum health points.
lives int Current number of lives.
score int Current score (Players only).
victim Character The character currently being attacked by this one.
attacker Character The character currently attacking this one.
damage_taken int Amount of damage received in the last hit.
combo_count int Current hit combo count.
combo_reset_timer int Frames remaining before the combo resets.
is_hitting_someone bool True if an attack is currently connecting.
is_vaulting bool True if the character is vaulting over an opponent.
is_blocking bool True if the character is in a blocking state.

Methods¶

check_being_hit_or_grabbed()¶

Checks if the character is currently being hit or grabbed.

check_being_hit()¶

Checks if the character is currently being hit.

check_being_grabbed()¶

Checks if the character is currently being grabbed.

check_hitting_entities()¶

Checks if the character's attack hitboxes are colliding with any victims.

try_grab(target, check_vicinity, enforcement, snap_to_victim)¶

Attempts to grab another character. - target: The character ID or object to grab. - check_vicinity: If true, only grabs if nearby.

set_animation(id)¶

Changes the character's current animation.

play_sound(pack_name, sound_name, channel)¶

Plays a sound effect from a specific content pack.

spawn_explosion(offset_x, offset_y, mode, direction)¶

Spawns an explosion effect at the specified offset.

try_spawn_shuriken(off_x, off_y, vel_x, vel_y, lifetime)¶

Specifically for ninja characters, spawns a shuriken projectile.

distribute_score(score)¶

Distributes a specified score among all active players.

distribute_score_based_on_damage(damage)¶

Calculates and distributes score based on damage dealt to an enemy.
