Player API¶

A Player is a specialized Character that is controlled by a human input. It contains and extends all properties and methods from the Character API.

Properties¶

Property Type Description
is_alive bool True if the player has lives remaining or health.
player_index int The zero-based index of the player.
input_state int The raw bitmask of the current input state.

Button Held States (Continuous)¶

These properties are true as long as the player is holding the respective button down.

Property Type Description
is_holding_up bool True if Up is being held.
is_holding_down bool True if Down is being held.
is_holding_left bool True if Left is being held.
is_holding_right bool True if Right is being held.
is_holding_forward bool True if holding the direction the player is facing.
is_holding_backward bool True if holding the direction opposite to facing.
is_holding_button_a bool True if the Special button (A) is held.
is_holding_button_b bool True if the Attack button (B) is held.
is_holding_button_c bool True if the Jump button (C) is held.
is_holding_button_x bool True if the X button is held.
is_holding_button_y bool True if the Y button is held.

Button Pressed States (Just Pressed)¶

These properties are true only on the exact frame the button was pressed.

Property Type Description
pressed_up bool True if Up was just pressed.
pressed_down bool True if Down was just pressed.
pressed_forward bool True if the forward direction was just pressed.
pressed_backward bool True if the backward direction was just pressed.
pressed_button_a bool True if the Special button (A) was just pressed.
pressed_button_b bool True if the Attack button (B) was just pressed.
pressed_button_c bool True if the Jump button (C) was just pressed.

Timers & Combo States¶

Property Type Description
back_attack_pressed bool True if a back attack was triggered (usually B+C).
is_special_attack_ready bool True if the player has enough health/meter to use a special.
special_spam_timer int Buffer timer for special attack button.
attack_spam_timer int Buffer timer for normal attack button.
jump_spam_timer int Buffer timer for jump button.
hold_direction_frames int Counter of frames a direction button has been held.

Methods¶

has_input_combo(combo_string)¶

Checks if the player recently inputted a specific sequence of commands (e.g., "F, F, B"). - combo_string: The input string to check for.

check_grab_while_moving()¶

Handles logic for grabbing enemies while walking into them.

safe_land()¶

Returns the player to a standing state, typically used after recovering from a hit in the air.

respawn()¶

Respawns the player in the current scene.
